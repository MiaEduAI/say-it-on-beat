# 🎤 Say It On Beat

**课堂节奏词汇游戏生成器 | Classroom Vocabulary Rhythm Game Generator**

灵感来自 TikTok 火遍全球的 Say It On Beat 挑战，改造成学科词汇复习工具。适用于任何学科、任何语言。

Inspired by the viral TikTok "Say It On Beat" challenge, adapted for classroom vocabulary review. Works with any subject, any language.

## 怎么用 How to Use

1. 打开网页 Open the page → [https://MiaEduAI.github.io/say-it-on-beat/](https://MiaEduAI.github.io/say-it-on-beat/)
2. 复制 Prompt → 粘贴到任何 AI（ChatGPT / 豆包 / Kimi / Claude）
3. 告诉 AI 你的学科和主题，AI 自动选词分组
4. 把结果粘贴回网页 → 生成游戏 → 投屏开玩！

## 4 种模式 Game Modes

- 🟢 经典 Classic — 按顺序走一遍
- 🔀 随机 Random — 随机跳着高亮
- 🔥 加速 Accelerate — 越来越快
- 🧠 记忆 Memory — 先看一遍，再凭记忆喊

## 特点 Features

- 零成本，不需要 API，不需要服务器
- 一个 HTML 文件，离线也能用
- 任何学科、任何语言、任何 AI 都适配
- 5 个 Level，速度递增，押韵分组
